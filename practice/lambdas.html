Practice problems: Lambda expressions

If you are studying for an exam, or you just want some extra practice with concepts in this class, you might find these practice problems helpful!

Easy

Question 1: Mul_by_num

Using a lambda expression, complete the mul_by_num function. This function should take an argument and return a one argument function that multiplies any value passed to it by the original number. Its body must be one line long:

def mul_by_num(num):
    """
    Returns a function that takes one argument and returns num
    times that argument.
    >>> x = mul_by_num(5)
    >>> y = mul_by_num(2)
    >>> x(3)
    15
    >>> y(-4)
    -8
    """

    "*** YOUR CODE HERE ***"
    return ______

    return lambda num2: num * num2

Medium

Question 2: Butch Cassidy and the Environment Diagram

Draw the environment diagram for the following code:

def blondie(f):
    return lambda x: f(x + 1)

tuco = blondie(lambda x: x * x)
angel_eyes = tuco(2)
