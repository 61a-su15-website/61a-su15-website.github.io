Question 4

double
scm> (deep-map double '(2 (3 4)))
(4 (6 8))

Assume that the input has no dotted (malformed) lists.

Hint: You can use the function list? to check if a value is a list.

(define (deep-map fn s)
  'YOUR-CODE-HERE
  nil
)

python3 ok -q deep-map
